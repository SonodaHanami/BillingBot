# BillingBot
记账bot

## 使用方法
克隆本仓库
```sh
git clone https://github.com/SonodaHanami/BillingBot.git
cd BillingBot
```
复制一份配置文件并修改
```sh
cp default-config.py config.py
```
安装docker和docker-compose
- https://docs.docker.com/engine/install/
- https://docs.docker.com/compose/install/

然后启动（debug去掉-d）
```sh
docker-compose up -d 
```
